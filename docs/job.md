# Job Functions

------------

## Shared Functions

##### Table for all informations about the Job

```lua
PLAYER:getJobTable()
```

##### Check if Player is a Trooper

```lua
PLAYER:IsTrooper()
```

##### Don't use it

```lua
PLAYER:GetRank()
```

##### Get TeamID

```lua
PLAYER:getTeamID()
```
