# Clothingstore Functions

------------

## Shared Functions

##### Register new Item to the store

```lua
CSTORE.RegisterNewItem( id, data )
```

## Datatable Arguments

| argument                     | description                              | 
|------------------------------|------------------------------------------|
| `ClothType`                  | Mask,Helm,Body                           |
| `Name`                       | Itemname                                 |
| `Sex`                        | 0 or 1                                   |
| `Faction`                    | 0,1,2 or false                           |
| `Model`                      | Modelpath                                |
| `Level`                      | Level                                    |
| `Price`                      | Price                                    |
